🚀 Getting Started

1. Clone the Repository

git clone https://github.com/innovateweb25/whatsapp-ai-auto-reply-bot.git
cd whatsapp-ai-auto-reply-bot

---

2. Setup Environment Variables

Go to the "app/" folder and create a ".env" file:

OPENROUTER_API_KEY=your_api_key_here
AI_MODEL=arcee-ai/trinity-large-preview:free
AI_NAME=Assistant

---

3. Start the Bot

chmod +x start.sh
./start.sh

---

4. Connect WhatsApp

- Open WhatsApp on your phone
- Go to Linked Devices
- Scan the QR code in terminal
- You're connected ✅

---

🧠 Customize AI Behavior

👉 Edit this file:

📂 "app/src/ai/SystemPrompt.js"

You can fully control how AI behaves:

- Business type
- Tone (friendly / professional)
- Language (Hinglish / English)
- Rules & restrictions
- Response style

⚙️ How It Works

1. User sends message on WhatsApp
2. Bot receives message
3. AI generates reply using SystemPrompt
4. Bot sends response automatically
